host: IP Address of login server

port: Port of login server

dbhost: IP Address of MySQL/db4o server

         or Path of saving for dat DB mode(example: ".\" or "c:\save\".)

dbname: Database name in MySQL database

dbport Port for MySQL/db4o server.  Only valid for MySQL or DB4O mode

 

ifSQL: this field is a little bit confusing, this is actually DB Mode, it has 3 options

        0: DB4O - not supported in current version of Saga

        1: MySQL

        2: Dat  - A file based saving solution, it's the simplest to use.

 

dbuser: User for DB server. Only valid for MySQL or DB4O mode

dbpass: Password for DB server.  Only valid for MySQL or DB4O mode

 

loginserverhost: IP Address of login server

loginserverport: Port of login server

loginpass:Password of login server

 

worldname: Server Name
