This file is used for defining bonuses of items/skills

 

example:

<addition>

   <id>601</id>

   <Unknown1>1</Unknown1>

   <Unknown2>1</Unknown2>

   <Func1>MoveSpeed</Func1>

   <Func2>Def</Func2>

   <Func3>FireResist</Func3>

   <Func4>IceResist</Func4>

   <Func5>WindResist</Func5>

   <Func6>HolyResist</Func6>

   <Func7>0</Func7>

   <Func8>0</Func8>

   <Func9>0</Func9>

   <Func10>0</Func10>

   <Value1>20300</Value1>

   <Value2>20150</Value2>

   <Value3>20150</Value3>

   <Value4>20150</Value4>

   <Value5>20150</Value5>

   <Value6>20150</Value6>

   <Value7>0</Value7>

   <Value8>0</Value8>

   <Value9>0</Value9>

   <Value10>0</Value10>

</addition>

 

id: ID of this bonus

Func1-10: Function of this bonus, see here for details

Value1-10: Values of each functions, greater than 20000 or less than -20000 is for percentage(20130 stands for 13% for example), anything else stands for points
